Create and edit vector graphics in your browser.

Drag and drop an image (JPG, PNG, WebP…) or SVG. Bitmap import is the default so your image appears on the canvas; uncheck to trace vectorize instead.

After a Bitmap only import, pick Pixel in the toolbar and draw on the image with the current HEX color.

Doppleit Vector — help

            
Tools: Pen (1), Rectangle (2), Circle (3), Text (4), or Select (S) to draw or edit.

            
Layers: Add, rename (double-click), drag to reorder, or group. Toggle visibility/lock.

            
Timeline: Previous/Next for version history when enabled.

            
Color: Wheel, sliders, or HEX/RGB/HSL.

            
Export: SVG, PNG, JPEG, or save/load project files.

            
UX: Shift for uniform scale, Alt to duplicate, drag guides from rulers.

            
Shortcuts: Undo (Ctrl+Z), Redo (Ctrl+Y), Delete, Escape, tools 1–4 and S.

            
Mobile: Pinch to zoom, touch controls below the canvas when shown.
